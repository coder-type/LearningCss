  • Linear Equation: An equation of the form a‚ÇÅx‚ÇÅ + a‚ÇÇx‚ÇÇ + ... + a‚Çôx‚Çô = b.
  • System of Linear Equations: A collection of one or more linear equations involving the same set of variables.
  • Types of Solutions:
    • Unique Solution
    • No Solution (Inconsistent)
    • Infinitely Many Solutions (Dependent)
  • Matrix Form: Represented as AX = B, where A is the coefficient matrix, X is the variable matrix, and B is the constant matrix.
  • Augmented Matrix: Combined matrix [A | B] used for solving the system.
  • Row Operations:
    • Row swapping
    • Row multiplication
    • Row addition/subtraction
  • Row Echelon Form (REF) and Reduced Row Echelon Form (RREF): Forms used to simplify matrices.
  • Gaussian Elimination: Solves linear systems by converting to REF and back-substitution.
  • Gauss-Jordan Elimination: Further reduces to RREF for direct solutions.
  • Homogeneous System: All constants are zero (i.e., AX = 0).
  • Consistency: A system is consistent if it has at least one solution.
