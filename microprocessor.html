MICROPROCESSOR NOTES

Chapter 1: Introduction to Microprocessor

Definition

A microprocessor is an integrated circuit that contains all the functions of a central processing unit of a computer. It is a semiconductor chip (like silicon) with a combination of transistors.

It is an electronic component that performs the instructions and tasks involved in computer processing which is the central unit and manages the logical instructions passed to it. In short, it processes arithmetic and logical operations to provide desired output.

Evolution/History of Microprocessor

  • Mancian E. Hubb is considered the father of the microprocessor.
  • The first commercial microprocessor was INTEL 4004 in 1971, with 2300 transistors (4-bit). Designed for calculators.
  • Next came INTEL 8008 (3500 transistors), then INTEL 8080 (4000 transistors), and ZILOG Z80 (6000 transistors).
  • The first 8-bit microprocessor was INTEL 8085.
  • Followed by INTEL 8086 (16-bit).
  • Evolution continued up to INTEL Pentium III (9.5 million transistors).
  • Modern microprocessors include the i3, i5, and i7 series.

Components of Microprocessor

The basic parts of a microprocessor are CPU, Bus, and Memory.

CPU (Central Processing Unit)

Fabricated as a Very Large Scale Integrated circuit (VLSI) with these parts:

  • Instruction Registers (IR): Holds instructions to be executed.
  • Decoder: Converts instructions to machine language and sends to ALU.
  • ALU (Arithmetic Logic Unit): Performs arithmetic, logical, memory register, and program sequencing operations.
  • Register: Holds intermediate results during program processing.

Bus

The fine thin lines connecting different internal parts of the microprocessor. There are three types:

1. Data Bus

  • Carries data to and from memory.
  • Bidirectional (data flows both directions).
  • Width equals word length (e.g., 8-bit in 8085 microprocessor).
  • For 8085 microprocessor, ranges from 00H to FFH (hexadecimal).

2. Address Bus

  • Unidirectional (carries address only from CPU to memory or I/O port).
  • Carries address of a memory location or I/O port.

3. Control Bus

  • Carries control signals like clock signals, interrupt signals, or ready signals.
  • Bidirectional.
  • Used to generate timing and control signals.
  • Control signals include:
    • Memory read
    • Memory write
    • I/O read
    • I/O write
  • Example: If a control bus wire is low (low electricity), memory is read; if high, memory is written.

Memory

Microprocessor has two types of memory:

  • RAM (Random Access Memory):
    • Volatile memory (loses data when power is off).
    • Working/runtime memory of the computer.
  • ROM (Read Only Memory):
    • Non-volatile memory (retains data without power).
    • Pre-programmed by manufacturer with essential data.

Control and Timing Unit

Provides timing and control signals to the microprocessor to perform various operations. It operates with reference to clock signals.

Control Signals

  • ALE (Address Latch Enable): Synchronizes components of microprocessor.
  • WR (Write): Used for writing operations (active low).
  • RD (Read): Used for reading operations (active low).

Status Signals

Three status signals used in microprocessor: S0, S1, and IO/M'. They change status according to provided inputs.

IO/M' S0 S1 Data Bus Status (Output)
0 0 0 Halt
0 0 1 Memory write
0 1 0 Memory read
1 0 1 IO write
1 1 0 IO read
0 1 1 Opcode fetch
1 1 1 Interrupt Acknowledge

Applications of Microprocessors

  1. Instrumentation:
    • Frequency counters, function generators, spectrum analyzers
    • Medical instrumentation
  2. Control:
    • Home appliances (microwave ovens, washing machines)
    • Industrial control (speed, pressure, temperature with transducers)
  3. Communication:
    • Telephone industry (digital sets, exchanges, modems)
    • Television, satellite communication
    • Railway/air reservation systems
    • LAN and WAN communication
  4. Consumer Electronics:
    • Toys, entertainment equipment
    • Calculators, accounting systems
    • Traffic light control
    • Military applications
    • Complex industrial controllers
