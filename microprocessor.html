MICROPROCESSOR NOTES

Chapter 1: Introduction to Microprocessor

Definition

A microprocessor is an integrated circuit that contains all the functions of a central processing unit of a computer. It is a semiconductor chip (like silicon) with a combination of transistors.

It is an electronic component that performs the instructions and tasks involved in computer processing which is the central unit and manages the logical instructions passed to it. In short, it processes arithmetic and logical operations to provide desired output.

Evolution/History of Microprocessor

  • Mancian E. Hubb is considered the father of the microprocessor.
  • The first commercial microprocessor was INTEL 4004 in 1971, with 2300 transistors (4-bit). Designed for calculators.
  • Next came INTEL 8008 (3500 transistors), then INTEL 8080 (4000 transistors), and ZILOG Z80 (6000 transistors).
  • The first 8-bit microprocessor was INTEL 8085.
  • Followed by INTEL 8086 (16-bit).
  • Evolution continued up to INTEL Pentium III (9.5 million transistors).
  • Modern microprocessors include the i3, i5, and i7 series.

Components of Microprocessor

The basic parts of a microprocessor are CPU, Bus, and Memory.

CPU (Central Processing Unit)

Fabricated as a Very Large Scale Integrated circuit (VLSI) with these parts:

  • Instruction Registers (IR): Holds instructions to be executed.
  • Decoder: Converts instructions to machine language and sends to ALU.
  • ALU (Arithmetic Logic Unit): Performs arithmetic, logical, memory register, and program sequencing operations.
  • Register: Holds intermediate results during program processing.

Bus

The fine thin lines connecting different internal parts of the microprocessor. There are three types:

1. Data Bus

  • Carries data to and from memory.
  • Bidirectional (data flows both directions).
  • Width equals word length (e.g., 8-bit in 8085 microprocessor).
  • For 8085 microprocessor, ranges from 00H to FFH (hexadecimal).

2. Address Bus

  • Unidirectional (carries address only from CPU to memory or I/O port).
  • Carries address of a memory location or I/O port.

3. Control Bus

  • Carries control signals like clock signals, interrupt signals, or ready signals.
  • Bidirectional.
  • Used to generate timing and control signals.
  • Control signals include:
    • Memory read
    • Memory write
    • I/O read
    • I/O write
  • Example: If a control bus wire is low (low electricity), memory is read; if high, memory is written.

Memory

Microprocessor has two types of memory:

  • RAM (Random Access Memory):
    • Volatile memory (loses data when power is off).
    • Working/runtime memory of the computer.
  • ROM (Read Only Memory):
    • Non-volatile memory (retains data without power).
    • Pre-programmed by manufacturer with essential data.

Control and Timing Unit

Provides timing and control signals to the microprocessor to perform various operations. It operates with reference to clock signals.

Control Signals

  • ALE (Address Latch Enable): Synchronizes components of microprocessor.
  • WR (Write): Used for writing operations (active low).
  • RD (Read): Used for reading operations (active low).

Status Signals

Three status signals used in microprocessor: S0, S1, and IO/M'. They change status according to provided inputs.

IO/M' S0 S1 Data Bus Status (Output)
0 0 0 Halt
0 0 1 Memory write
0 1 0 Memory read
1 0 1 IO write
1 1 0 IO read
0 1 1 Opcode fetch
1 1 1 Interrupt Acknowledge

Applications of Microprocessors

  1. Instrumentation:
    • Frequency counters, function generators, spectrum analyzers
    • Medical instrumentation
  2. Control:
    • Home appliances (microwave ovens, washing machines)
    • Industrial control (speed, pressure, temperature with transducers)
  3. Communication:
    • Telephone industry (digital sets, exchanges, modems)
    • Television, satellite communication
    • Railway/air reservation systems
    • LAN and WAN communication
  4. Consumer Electronics:
    • Toys, entertainment equipment
    • Calculators, accounting systems
    • Traffic light control
    • Military applications
    • Complex industrial controllers
Chapter 2: Microprocessor Architecture and Microcomputer Systems

Definition

The microprocessor architecture refers to the internal structure and organization of a microprocessor, detailing how it processes instructions and interacts with memory and I/O devices. In microcomputer systems, the microprocessor serves as the central unit that manages and controls all operations.

8085 Microprocessor Architecture

The 8085 microprocessor is an 8-bit processor with a 16-bit address bus, capable of addressing 64KB of memory. Its architecture comprises several key components:

1. Arithmetic and Logic Unit (ALU)

  • Performs arithmetic operations like addition and subtraction.
  • Executes logical operations such as AND, OR, and XOR.
  • Sets or resets flags based on the result of operations.

2. Register Array

  • Includes six general-purpose registers: B, C, D, E, H, and L.
  • Registers can be paired as BC, DE, and HL for 16-bit operations.
  • Accumulator (A) is used for arithmetic and logic operations.

3. Program Counter (PC) and Stack Pointer (SP)

  • Program Counter (PC): Holds the address of the next instruction to be executed.
  • Stack Pointer (SP): Points to the top of the stack in memory, used during subroutine calls and interrupts.

4. Instruction Register and Decoder

  • Instruction Register: Temporarily holds the current instruction fetched from memory.
  • Decoder: Interprets the instruction and signals the necessary operations to be performed.

5. Timing and Control Unit

  • Generates control signals to synchronize operations of the microprocessor.
  • Manages the flow of data between the microprocessor and peripherals.

6. Interrupt Control

  • Handles interrupt signals, allowing the microprocessor to respond to external events.
  • Supports multiple interrupt lines with varying priorities.

7. Serial I/O Control

  • Manages serial communication through SID (Serial Input Data) and SOD (Serial Output Data) lines.

8. Address and Data Buses

  • Address Bus: 16-bit unidirectional bus used to address memory locations.
  • Data Bus: 8-bit bidirectional bus used to transfer data between the microprocessor and memory or I/O devices.

Microcomputer System Organization

A microcomputer system integrates the microprocessor with memory and I/O devices to perform computing tasks. The primary components include:

1. Memory

  • RAM (Random Access Memory): Volatile memory used for temporary data storage during program execution.
  • ROM (Read-Only Memory): Non-volatile memory containing permanent instructions or data.

2. Input/Output Devices

  • Allow the microcomputer to interact with the external environment.
  • Examples include keyboards, displays, printers, and communication interfaces.

3. System Bus

  • Comprises the address bus, data bus, and control bus.
  • Facilitates communication between the microprocessor, memory, and I/O devices.

4. Clock Generator

  • Provides the necessary clock signals to synchronize operations within the microcomputer system.

5. Power Supply

  • Supplies the required electrical power to all components of the microcomputer system.

Applications of Microprocessor-Based Systems

  1. Embedded Systems: Used in appliances, automobiles, and industrial machines for control functions.
  2. Personal Computers: Serve as the central processing unit in desktops and laptops.
  3. Communication Devices: Integral to mobile phones, routers, and modems.
  4. Instrumentation: Employed in medical devices, measurement instruments, and monitoring systems.
  5. Consumer Electronics: Found in televisions, gaming consoles, and smart home devices.
