  • Definition: A microprocessor is a multipurpose, programmable, clock-driven, register-based electronic device that reads binary instructions from a storage deivce called memory and accepts binary data as input and processes data according to those instructions and then provides results as output.
  • History: First microprocessor was Intel 4004 (4-bit) developed in 1971.
    • Marcian.E. Hubb was the creator and also known as father of microprocessor. The first commerical microprocessor came in 1971, which was a 4-bit microprocessor having 2300 transistors. It was basically designed for calculators at that time.
    • The next intel microprocessor was INTEL 8008 having 3500 transistors after this, INTEL 8080 having 4000 transistors and ZILOG Z80 having 6000 transistors.
    • The 4-bit intel microprocessor was continued and the first 8-bit microprocessor was developed which is intel 8085 which is also important for understanding the core of this subject.
    • Similarly, the next microprocessor was the intel 8086 microprocessor.
    • And then, the evolution of microprocessor continued into PENTIUM III having 9.5 million transistors, here are some of the pentium processors like Pentium, Pentium Pro, Pentium MMX, Pentium Dual-Core.
    • After the Dual-Core Pentium, Intel's primary mainstream processor family became the Core i series, which includes Core i3, Core i5, Core i7, and Core i9.
  • Components of Microprocessor
  • 1. Microprocessor (CPU)

    • The brain of the system. Executes instructions, performs arithmetic/logical operations, controls data flow. Examples: Intel 8085, 8086, etc.

  • Memory
    • RAM (Random Access Memory)
    • Temporary storage (volatile).
    • Stores data and programs during execution.
    b. ROM (Read-Only Memory) Permanent storage (non-volatile). Stores boot code or firmware. 3. I/O Devices (Input/Output) Used for interaction with the external world. Input devices: Keyboard, switches, sensors. Output devices: Display, LEDs, actuators.
  • Basic Features:
    • ALU (Arithmetic Logic Unit)
    • Registers
    • Control Unit
    • Clock and Reset
  • Important Terms:
    • Instruction Set: Set of commands the processor can execute
    • Bus: Pathway for data (Data Bus, Address Bus, Control Bus)
  • 8085: It is an 8-bit microprocessor developed by Intel in 1976.
  • Features of 8085:
    • 8-bit data bus
    • 16-bit address bus
    • 74 instructions and 246 opcodes
    • Single +5V supply
    • Clock speed of 3 MHz
  • Functional Units:
    • Accumulator
    • ALU
    • General Purpose Registers (B, C, D, E, H, L)
    • Program Counter (PC)
    • Stack Pointer (SP)
    • Instruction Register and Decoder
    • Timing and Control Unit
