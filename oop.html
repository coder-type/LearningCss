ðŸ“— Chapter 2: Classes and Objects
  • Class: A blueprint for creating objects. It defines properties and behaviors.
  • Object: An instance of a class.
  • Access Modifiers:
    • private: Accessible only within the class.
    • public: Accessible from outside the class.
    • protected: Accessible in the same class and subclasses.
  • Constructor: A special method used to initialize objects.
  • Destructor: A method that is automatically called when an object is destroyed (in some languages).
  • Example in C++: 
    
class Student {
  public:
    string name;
    int age;

    void display() {
      cout << "Name: " << name << ", Age: " << age << endl;
    }
};
