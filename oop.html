Object-Oriented Programming (OOP) - Notes

ðŸ“˜ Introduction To Object Oriented Programming
  • Definition: OOP is a programming paradigm based on the concept of "objects", which can contain data and code to manipulate that data.
  • Key Concepts:
    • Class: A blueprint for creating objects.
    • Object: An instance of a class.
    • Encapsulation: Bundling data and methods that operate on the data within one unit.
    • Abstraction: Hiding complex implementation details and showing only the necessary features.
    • Inheritance: Mechanism by which one class can inherit properties and methods from another.
    • Polymorphism: Ability to process objects differently based on their data type or class.
  • Advantages of OOP:
    • Modularity: Code is organized into classes and objects, making it easier to manage and update.
    • Reusability: Classes and methods can be reused across different programs, reducing redundancy.
    • Scalability: OOP makes it easier to expand and maintain large codebases over time.
    • Maintainability: Since code is modular and organized, debugging and updating become simpler.
    • Data Security: Encapsulation protects data through controlled access.
  • Procedural Programming vs OOP:
    Aspect Procedural Programming Object-Oriented Programming
    Focus Functions and procedures Objects and classes
    Code Organization Linear and top-down Modular and hierarchical
    Data Handling Global data shared among functions Encapsulated within objects
    Reusability Limited code reuse High code reuse via inheritance
    Security Less secure, data is open More secure through encapsulation
    Examples C, Fortran C++, Java, Python (with OOP)
ðŸ“˜ Basic of C++ Programming

Introduction to C++

  • C++ is an extension of the C programming language with object-oriented features.
  • Supports both procedural and object-oriented programming paradigms.
  • Developed by Bjarne Stroustrup in early 1980s.

Structure of a C++ Program

  • Every C++ program has a main() function which acts as the entry point.
  • Basic structure includes header files, main function, and statements inside it.
#include <iostream>
using namespace std;

int main() {
  cout << "Hello, World!"; 
  return 0;
}

Variables and Data Types

  • Variables store data; must be declared with a data type before use.
  • Common data types: int, float, double, char, bool, string.
  • Example: 
    int age = 25;
float price = 99.99;
char grade = 'A';

Input and Output

  • Use cin for input and cout for output.
  • Example: 
    int number;
cout << "Enter a number: ";
cin >> number;
cout << "You entered: " << number;

Functions in C++

  • Functions are blocks of code that perform specific tasks.
  • Syntax: 
    return_type function_name(parameters) {
  // body
}
  • Example: 
    int add(int a, int b) {
  return a + b;
}

Classes and Objects

  • Class: Blueprint to create objects.
  • Object: Instance of a class.
  • Syntax for class: 
    class Car {
  public:
    string brand;
    int year;

    void display() {
      cout << "Brand: " << brand << ", Year: " << year << endl;
    }
};
  • Creating object and using: 
    Car car1;
car1.brand = "Toyota";
car1.year = 2020;
car1.display();

Constructors

  • Special functions to initialize objects.
  • Have same name as the class, no return type.
  • Example: 
    class Car {
  public:
    string brand;
    int year;

    Car(string b, int y) {  // Constructor
      brand = b;
      year = y;
    }
};

Access Specifiers

  • Define visibility of class members.
  • Types:
    • public - accessible from outside the class
    • private - accessible only inside the class
    • protected - accessible in derived classes
