Buses in a Microprocessor

What is a Bus?

A bus is a set of parallel lines used to transfer data, address, and control signals between components like the CPU, memory, and I/O devices. It acts like a digital highway.

Types of Buses

1. Data Bus

Transfers actual data. It is bidirectional. In 8085, it is 8-bit wide (AD0 to AD7).

2. Address Bus

Carries the address of memory or I/O location. It is unidirectional. In 8085, it is 16-bit wide (A8 to A15, AD0 to AD7).

3. Control Bus

Carries control and timing signals. Used to coordinate operations and signal what action to take (like RD̅, WR̅, ALE, etc.).

Bus Summary

Bus Type Width (8085) Direction Purpose
Data Bus 8 bits Bidirectional Transfer actual data
Address Bus 16 bits Unidirectional Specify memory/I/O address
Control Bus Varies Both Send control signals (RD̅, WR̅, etc.)

Example Operation

CPU reads data from memory:

  1. CPU places address on the address bus
  2. Control bus sends RD̅ signal to memory
  3. Memory puts data on the data bus
  4. CPU reads the data
